# 🚀 Deployment

This portfolio is deployed using **GitHub Pages**.

Steps:

1. Open repository **Settings**
2. Navigate to **Pages**
3. Select:

Branch → main
Folder → root

4. Save

Your site will be live at:

https://kskreddy2k7.github.io/kranthu-ai-portfolio/
